# Personal Policy Data Project v01
# Career-Linked Version（BOJ / DBJ接続強化版）
# Theme: 国の公共投資は地域の持続性にどう波及したか
# Case: 兵庫県中心（類似県比較）
# Tools: Python中心
# Goal: 実装型公共経済アナリストとしての能力証明

============================================================
0. プロジェクトの二重目的
------------------------------------------------------------
[研究目的]
  国の公共投資が地域の若年人口および産業構造に与えた
  長期的影響を実証的に明らかにする。

[キャリア目的]
  日本銀行・日本政策投資銀行で求められる
  「政策データ実装力」「構造理解力」「再現可能分析能力」を
  実証的成果物として提示できるようにする。

E. コード品質
  - 再現可能
  - モジュール化
  - README完備

============================================================
2. レポート設計（意味のある成果物）
------------------------------------------------------------

Output 1:
  「兵庫県の若年人口長期動向と全国比較（2000–2022）」
  → 基礎構造把握レポート

Output 2:
  「兵庫内部の若年人口分解：都市部と非都市部の差」
  → 地域内部異質性分析

Output 3:
  「類似県抽出の方法と妥当性」
  → データ駆動型地域分類の技術メモ

Output 4:
  「兵庫県の産業構造転換と人口動態の関係」
  → 構造分析レポート

Output 5（高度版）:
  「公共投資は兵庫の若年人口を押し上げたか」
  → 合成コントロール分析

すべて：
  ・図表中心
  ・Executive Summary 1ページ
  ・技術付録付き
  ・再現コードリンク明記
